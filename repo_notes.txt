**PROJECT STRUCTURE**

src/
  data/              # Data loading
    load_bbq.py      # BBQ dataset pairs
    load_pairs.py    # Unified loader (BBQ, MMLU, GSM-MC, SGXS)

  core/              # Core pipeline
    rm_utils.py      # Skywork RM wrapper
    layer_attack_direction.py  # LAT attack construction
    extract_directions.py      # Direction extraction
    pca_directions.py          # PCA decomposition
    label_components.py        # Auto-labeling

runners/             # Experiment runners
  run_core_sweep.py  # Main sweep
  run_eps_sweep.py   # Epsilon sweep
  run_label_all_pca.py

analysis/            # Post-hoc analysis
  analyze_robustness_sweep.py     # Robustness analysis
  analyze_pca_and_labels_sweep.py # PCA interpretation

utils/               # Helper scripts
  summarize_component_contributions.py
  summarize_pca_run.py
  concise_summarize_pca_labels.py
  pick_and_summarize_runs.py

**RESULTS STRUCTURE**

results/
  outputs/           # Direct pipeline outputs
    directions/      # Direction vectors (.pt)
    pca/             # PCA results (.pt)
    labels/          # Semantic labels (.jsonl)
    deltas/          # Delta tensors
  analysis/          # Derived analysis
    robustness/      # Robustness analysis outputs
    pca/             # PCA analysis outputs
    figs/            # Visualization figures
  data/              # Sweep/log data
    sweeps/          # Large sweep outputs
    logs/            # Pipeline logs

**PIPELINE FLOW**

Data Loading:
  src/data/load_pairs.py -> src/data/load_bbq.py

Direction Extraction:
  src/core/extract_directions.py (uses layer_attack_direction.py, rm_utils.py)
  Output: results/outputs/directions/direction_records_*.pt

PCA Interpretation:
  src/core/pca_directions.py
  Output: results/outputs/pca/pca_*.pt

Auto-Labeling:
  src/core/label_components.py
  Output: results/outputs/labels/component_labels_*.jsonl

Analysis:
  analysis/analyze_robustness_sweep.py -> results/analysis/robustness/
  analysis/analyze_pca_and_labels_sweep.py -> results/analysis/pca/

Main Runner:
  runners/run_core_sweep.py (orchestrates: extract -> pca -> label)
